# MERN
Mongo Express React Node

### Dependencies
Start mongo db server
```
mongod
```
Install npm modules
```
cd mern-api
npm install
cd mern-client
npm install
```

### Run Api Backend
```
cd mern-api
npm start
```

### Run React Client
```
cd mern-client
npm start
```
